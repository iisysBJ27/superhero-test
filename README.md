Функция, которая по API супергероа ищет самого высокого героя
по двум критериям: пол и наличие работы. К функции написаны автотесты.

1. Открыть терминал в папке проекта (там, где лежат файлы superhero.py и test_superhero.py).

2. Активировать виртуальное окружение, если оно ещё не активно. На Windows команда такая: venv\Scripts\activate

3. Установить библиотеки6 pip install requests pytest

4. Тесты запускаются комадной: pytest -v

Python 3.10 или выше
